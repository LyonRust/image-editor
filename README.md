# 简介

无界云图是<a href="https://www.h5ds.com" target="_blank">四川爱趣五科技</a>推出的一款开源的图片设计工具，使用
了`React Hooks`，`Typescript`，`Vite`，`Leaferjs`等主流的技术开发，开箱即用，如果想直接使用，请访问我们的在线版
本<a href="https://image.h5ds.com" target="_blank">无界云图</a>。

# 是否可商用？

项目采用 MIT 协议，可免费商用。

# 安装使用

```javascript

// 安装依赖
yarn install

// 启动项目
yarn dev

// 打包项目
yarn build

```
